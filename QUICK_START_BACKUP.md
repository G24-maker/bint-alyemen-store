## ⚙️ الإعدادات المتقدمة

### تغيير وقت النسخ الاحتياطي:

فتح `backend/server.js` وابحث عن:
```javascript
cron.schedule('0 2 * * *', async () => {
```

غيّر `'0 2 * * *'` إلى:
- `'0 0 * * *'` - منتصف الليل (12:00 AM)
- `'0 12 * * *'` - الظهيرة (12:00 PM)
- `'30 3 * * *'` - الساعة 3:30 AM
- `'0 */6 * * *'` - كل 6 ساعات

### حذف النسخ القديمة يدويًا:

```bash
# اعرض المجلد
ls -la backend/backups/

# احذف ملف معين
rm backend/backups/backup-2026-01-20-00-00-00.sqlite
```

## 🆘 حل المشاكل

### المشكلة: لا أرى النسخ الاحتياطية

**الحل:**
```bash
npm run backup:list
ls -la backend/backups/
```

### المشكلة: النسخ الاحتياطي لا يعمل

**الحل:**
```bash
# تحقق من node-cron
npm list node-cron

# اختبر النسخ اليدوي
npm run backup

# افحص السيرفر
npm run dev
```

### المشكلة: الاستعادة لم تنجح

**الحل:**
```bash
# أوقف السيرفر (اضغط Ctrl+C)

# تأكد من اسم الملف
npm run backup:list

# استعد من النسخة
node backup.js restore <filename>

# أعد تشغيل السيرفر
npm run dev
```
